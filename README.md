The Babel
====

## Description

Let's make the highest tower in the space.
No warehouse is game over.
Enemies try to break warehouse.
To save food,make the house and weapon.
Then,Pop can go to space!

Move           : Cursor Keys
Build House    : Z
Build Weapon   : X
Build Warehouse: C
Build Ladder   : V
